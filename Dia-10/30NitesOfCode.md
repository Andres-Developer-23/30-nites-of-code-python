## Día 10 de #30NitesofCode 🚀

```python
# Python viene con herramientas ya incorporadas
import math
import random
from datetime import datetime

print(math.sqrt(25))           # 5.0
print(random.randint(1, 10))   # número aleatorio
print(datetime.now())          # fecha y hora actual
```

**Conceptos cubiertos:**
- `import modulo` — importa un módulo completo.
- `from modulo import funcion` — importa solo lo necesario.
- `import modulo as alias` — importa con un alias.
- Módulos estándar útiles: `math`, `random`, `datetime`, `os`, `sys`.
- `if __name__ == "__main__"` — controla la ejecución del script.
- Crear tus propios módulos (archivos .py que importas).
