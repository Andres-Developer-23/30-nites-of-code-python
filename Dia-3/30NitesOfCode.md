## Día 3 de #30NitesofCode 🚀

```python
# La estructura de decisión básica
x = 15

if x > 10:
    print("x es mayor que 10")
elif x == 10:
    print("x es igual a 10")
else:
    print("x es menor que 10")
```

**Conceptos cubiertos:**
- `if` — ejecuta código si la condición es verdadera.
- `elif` — condición adicional si la anterior fue falsa.
- `else` — ejecuta código si ninguna condición anterior se cumple.
- Indentación (4 espacios) — Python usa sangría para definir bloques.
- Condiciones anidadas: un `if` dentro de otro `if`.
