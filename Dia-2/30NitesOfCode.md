## Día 2 de #30NitesofCode 🚀

```python
# Los operadores trabajan con valores fijos
print(10 + 3)     # 13
print(10 - 3)     # 7
print(10 * 3)     # 30
print(10 / 3)     # 3.333...
print(10 // 3)    # 3 (división entera)
print(10 % 3)     # 1 (módulo)
print(10 ** 3)    # 1000 (potencia)
print(10 > 3)     # True
```

**Conceptos cubiertos:**
- Operadores aritméticos: `+`, `-`, `*`, `/`, `//`, `%`, `**`.
- Operadores de comparación: `==`, `!=`, `>`, `<`, `>=`, `<=`.
- Operadores lógicos: `and`, `or`, `not`.
- `input()` — función para recibir datos del usuario (siempre devuelve str).
- Conversión de tipos con `int()` y `float()`.
