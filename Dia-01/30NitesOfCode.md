## Día 1 de #30NitesofCode 🚀

```python
# Así se ven los tipos de datos en Python
print("¡Hola, mundo! 🐍")        # str
print(42)                         # int
print(3.1416)                     # float
print(True)                       # bool
print(type("Python"))             # <class 'str'>
```

**Conceptos cubiertos:**
- `print()` — función para mostrar mensajes en la terminal.
- Variables — contenedores para almacenar datos con un nombre.
- Tipos de datos: `int` (entero), `float` (decimal), `str` (texto), `bool` (booleano).
- `type()` — función para conocer el tipo de una variable.
